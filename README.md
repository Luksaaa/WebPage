# Boutique Hotel Postar - Official Website

Welcome to the digital landing page for **Boutique Hotel Postar**, a premium, high-end accommodation experience located in Zadar, Croatia. This project highlights modern web development techniques and elegant user interface design tailored for the hospitality industry.

## 🏨 Purpose of the Website
The primary goal of this project is to provide a seamless and visually appealing online presence for Boutique Hotel Postar. It allows potential guests to explore the hotel's luxury offerings before they even arrive.

Key objectives of the site:
- **Showcase Accommodations**: Detailed display of comfort levels and pricing for different room types.
- **Catering Services**: Overview of gourmet food packages and dining levels.
- **Geographic Exploration**: Integration of interactive maps to highlight the hotel's prime location on the Adriatic coast.
- **Brand Identity**: Establishing a sleek and modern look that matches the hotel's 15 luxury units.
- **Full Responsiveness**: Every element on the website is designed to be fully responsive, ensuring a perfect user experience across all devices, including desktops, tablets, and smartphones.

## ✨ Features
- **Modern & Fully Responsive Design**: Every single element is optimized for desktops, tablets, and mobile phones.
- **Intuitive Navigation**: Easy access to the Rooms, Catering, and Location sections.
- **Room Showcase**: High-resolution imagery and pricing for Comfort, President, and Premium suites.
- **Interactive Map**: Built-in Google Maps iframe for real-time location tracking in Zadar.
- **Catering Packages**: Clear breakdown of Premium, Medium, and Light catering tiers.
- **Comprehensive Footer**: Quick links to company info, help center, and social media icons (Instagram, Twitter, Facebook, LinkedIn).

## 🛠️ Built With
- **HTML5**: For semantic structure and content organization.
- **CSS3**: For custom typography (Montserrat), vibrant layouts, and hover effects.
- **Google Maps API**: Integrated map section for location services.

## 🚀 How to View the Project
To view the website locally:
1. Clone or download this repository.
2. Navigate to the `SJW-PROJEKT` directory.
3. Open `index.html` in any modern web browser (Chrome, Safari, Firefox, Edge).
